\section{公有函数设计}
\subsection{computeBarrierSecondDerivativeFree（Barrier 二阶导数自由项）函数设计}
\subsubsection{函数需求}
\par 本函数实现 \textbf{Barrier 二阶导数自由项}。其在 CBF 自动驾驶安全控制流水线中承担如下职责：求 Barrier 二阶导数中与控制 u 无关的部分（用于 HOCBF 仿射约束装配）。
\par 头文件路径：\texttt{include/cpp/computeBarrierSecondDerivativeFree/computeBarrierSecondDerivativeFree.hpp}；源文件路径：\texttt{src/cpp/computeBarrierSecondDerivativeFree/computeBarrierSecondDerivativeFree.cpp}。
\subsubsection{算法设计}
\par 本函数核心数学表达式如下：
\begin{equation}
\ddot B|_{free} = 2(v_{rx}^2 + v_{ry}^2) + 2(dx \cdot a_x^{ob} + dy \cdot a_y^{ob})
\end{equation}
\par 实现步骤：
\begin{enumerate}
  \item \textbf{vel\_sq}: velSq = vRx\textasciicircum{}2 + vRy\textasciicircum{}2
  \item \textbf{ob\_acc}: obAccTerm = dx*ax + dy*ay
  \item \textbf{sum}: ddBFree = 2*velSq + 2*obAccTerm
\end{enumerate}

\subsubsection{函数接口}
\par 输入参数列表：
\begin{itemize}
  \item \texttt{dxEgo} (double) -- 纵向相对位置
  \item \texttt{dyEgo} (double) -- 横向相对位置
  \item \texttt{vRxEgo} (double) -- 纵向相对速度
  \item \texttt{vRyEgo} (double) -- 横向相对速度
  \item \texttt{axObEgo} (double) -- 障碍物绝对加速度 ego-x 分量
  \item \texttt{ayObEgo} (double) -- 障碍物绝对加速度 ego-y 分量
\end{itemize}
\par 输出参数列表：
\begin{itemize}
  \item \texttt{return} (double) -- 二阶导数自由项 ddBFree
\end{itemize}
